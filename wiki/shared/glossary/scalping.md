# Scalping

Trading style that pairs a back and a lay on the same runner to lock profit from a price move regardless of race outcome. In `betfair_env` this is `scalping_mode`. Proper scalping requires asymmetric stakes ([[equal-profit-pair-sizing]]); equal stakes only lock when prices are symmetrical around 1.0. Stub — link from notes that reference it. [[glossary]]
