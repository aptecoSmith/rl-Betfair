# Glossary — acronyms & domain terms

The acronym dictionary. Each entry under `shared/glossary/` is a lightweight node so that a doc naming an
acronym (PPO, EW, LTP, …) links through a shared node to every other doc that uses it — the acronyms are
**connective tissue, not noise**. During ingest, `wiki_tool.py discover` shows these as EXISTING
("link these"); add the acronym wiki-link in the citing note (e.g. link `ppo`) rather than
`entity-skip`-ping it. New acronyms get a stub here as they first appear.
